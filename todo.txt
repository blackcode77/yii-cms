Реализовать простой проект на yii2, что-то типа музыкального сайта. Строить на основе yii2 шаблона advanced.
Дизайн не важен - используем стандартные виджеты (гриды, инпуты)  бутрстрапа из базовой поставки yii2.

Используем 3 модели данных:

Исполнитель
Имя
Год рождения
Альбом
Связь с исполнителем
Название
Год
Песня
Связь с альбомом
Длительность
Номер по порядку
Пользователь (из стандартной поставки)

Реализуем базовые CRUD контроллеры на все модели, но логику всю выносим в отдельные классы (по типу как Domain Driven Design, в Active Record оставляем только валидацию данных).
Пишем юнит тесты на классы логики.
Используя стандартный RBAC реализуем три вида прав - пользователь, модератор, админ. Пользователь только смотрит, модератор может только редактировать, без удаления. Администратор может все.

Не обязательно писать все 3 контроллера, можно написать один, но нормально + тесты к логике. Надо показать или не показать умение писать тесты, выносить логику в отдельные сущности, использовать встроенные компоненты yii2, такие как RBAC, formatter и прочие.
